Windowsでは、画像ファイルを全選択して「名前を変更」をし、1つ目の画像を「1.jpg」とすると自動的に他の画像も変更できます。
On Windows, you can automatically change the other images by selecting all the image files, doing "Rename", and setting the first image as "1.jpg".

LinuxやMacの人はコマンドを打って適当に解決してください。
If you are on Linux or Mac, please type the command and solve the problem appropriately.
